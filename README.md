## 📌 Specialization Overview
The specialization is divided into three core courses, transitioning from classical supervised learning to advanced neural networks and unsupervised systems.

### 1. Supervised Machine Learning: Regression and Classification
* **Linear Regression:** Cost functions = 损失函数, gradient descent = 梯度下降, and multi-feature models.
* **Logistic Regression:** Binary classification, sigmoid functions, and decision boundaries.
* **Overfitting:** Regularization = 正则化 techniques to improve model generalization.



### 2. Advanced Learning Algorithms
* **Neural Networks:** Building and training models with TensorFlow for multi-class classification.
* **Decision Trees:** Tree ensembles, random forests, and XGBoost.
* **Advice for ML:** Bias vs. Variance = 偏差与方差, error analysis, and iterative development.



### 3. Unsupervised Learning, Recommenders, Reinforcement Learning
* **Clustering & Anomaly Detection:** K-means and Gaussian distribution for fraud detection.
* **Recommender Systems:** Collaborative filtering and content-based deep learning.
* **Reinforcement Learning:** Deep Q-Learning and state-action value functions.
